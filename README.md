# aie -- The Artificial Intelligence Eye

## Installation

### Directly

```shell
git clone https://www.github.com/SkirOwen/aie.git
cd aie
# Install
pip install -e .
```

### Anaconda Virtual Env

In the directory, after cloning the repository
```shell
conda create -n aie python=3.8
conda activate aie
pip install -e .
```
